## Color Guessing Game

Light up color game that accepts button input to pick the correct answer, with blinking and audio feedback.

#### Components

- Arduino Uno
- 10k ohm resisters (6)
- mosfets (3)
- RGB LED strip
- buttons (6)

#### Pins

- 2 = button 1
- 3 = button 2
- 4 = button 3
- 5 = LED red
- 6 = LED green
- 7 = button 4
- 8 = speaker
- 9 = LED blue
- 10 = button 6
- 11 = button 5
